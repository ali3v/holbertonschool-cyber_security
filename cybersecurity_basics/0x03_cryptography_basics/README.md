# 0x03. Cryptography Basics

## Description
This project introduces the fundamental concepts of cryptography within the field of cybersecurity. Cryptography is the practice and study of techniques for secure communication in the presence of third parties.

The main objective of this repository is to explore the basics of data protection, understanding the differences between encryption and hashing, and learning how to utilize standard auditing tools.

## Learning Objectives
By the end of this project, the expected learning outcomes are:

* **Core Concepts:** Understanding what cryptography is and its importance in cybersecurity.
* **Mechanisms:**
    * Difference between **Encryption** and **Decryption**.
    * Types of cryptography (Symmetric vs. Asymmetric).
    * Understanding **Hash Algorithms** (e.g., SHA, MD5).
* **Tools & Applications:**
    * Introduction to **John the Ripper** for password auditing.
    * Introduction to **hashcat** for advanced hash cracking.
    * Understanding practical applications of cryptography in securing digital assets.
